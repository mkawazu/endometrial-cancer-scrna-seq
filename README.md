# scRNA-seq Analysis Code for peer review

## Overview
This repository contains the analysis code for single-cell RNA sequencing (scRNA-seq) data described in the manuscript.

## Notebooks
- `01_clustering.ipynb`: Preprocessing, quality control, 
  dimensionality reduction, clustering, and cell type annotation
- `02_summary.ipynb`: Downstream analyses and figure generation

## Usage
Run notebooks in the following order:
1. 01_clustering.ipynb
2. 02_summary.ipynb
